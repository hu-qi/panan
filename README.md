# 磐安县开咖啡馆可行性数据报告

浙江省金华市磐安县开咖啡馆的市场可行性分析，包含人口、消费、旅游、竞品数据 + Python 量化模型。

## 快速开始

```bash
python3 data_collector.py
```

输出：客群估算 + 3种情景收入预测 + SWOT + JSON/CSV数据导出。

## 文件结构

```
panan/
├── data_collector.py                     # Python 采集分析脚本（含全部原始数据）
├── panan_data.json                       # 结构化 JSON 数据（8 类数据源）
├── panan_data.csv                        # 时间序列 CSV
├── 磐安县咖啡馆可行性数据报告.md            # 完整报告
└── README.md
```

## 数据来源

| # | 内容 | 来源 |
|---|------|------|
| 1 | 人口/收入/GDP | 磐安县统计局《2024/2025年统计公报》 |
| 2 | 第七次人口普查 | 磐安县人民政府（2021.5.21） |
| 3 | 年龄结构 | 磐安县慢性病及社会影响因素报告（2018-2022） |
| 4 | 消费/餐饮增速 | 磐安县经济运行报告（2025.7） |
| 5 | 高铁客流 | 磐安广电公众号、中新网 |
| 6 | 咖啡竞品 | 携程/大众点评 |
| 7 | 浙江省对照 | 浙江省统计局、浙江省民政厅 |

## 核心结论

- **常住人口** 17.9 万，自然增长率 -2.24‰（负增长）
- **老龄化** 23.44%（60 岁以上），核心咖啡客群约 5,000-7,700 人
- **收入** 人均可支配收入 42,915 元（浙江平均的 64%）
- **消费** 限上社消增速 32.8% 全市第一，餐饮增速 20.1%
- **交通** 杭温高铁 2024 年通车，首周末接发 5,000 人次
- **竞品** 已有 4 家"村咖"，县城城区空白
- **中性方案** 日均 90 杯、客单价 22 元，约 10 个月回本
